## How to use this file

Each slide is written with a title, layout note, content, and speaker notes.
Build your presentation deck from this file. The content is the source of
truth — if you update the deck, update this file to match and bump the
version number.

Speaker notes are written in your voice — conversational and plain.
They are not a script to read verbatim but a guide for what to cover
and how to frame it.
